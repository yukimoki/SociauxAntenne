Lancer les scripts pour obtenir toutes les tables.

creationDossierStatistiques.py permet d'avoir les dossiers dans lesquels seront sauvegardés les graphiques
Il faut lancer creationDossierStatistiques.py avant de lancer les deux autres scripts pour ne pas avoir à créer les fichiers vous même et avoir des erreurs.

statGen.py créée deux graphiques dans le dossier /statistiques/emetteur_population_support/StatToutOperaturConfondu.
Le premier graphique, statPopGenMax.png représente la génération maximale d'émetteur de chaque support sans prendre en compte l'opérateur.
le second, statPopGenTous.png représente toutes les génération de chaque émetteur présent sur chaque support sans prendre en compte l'opérateur.

statGenParOperateur.py créée 18 graphiques.

Il créée dans le dossier /statistiques/emetteur_population_support/StatParOperateur 8 graphiques.
Les 8 graphiques sont nommé de manière similaires aux graphiques créée par statGen.py mais en tenant compte de l'opérateur.
Chaque graphique est proportionnel à ses données.
statPopgenMax[Opérateur].png représente la génération maximale par émetteur de chaque support de l'opérateur [Opérateur].
statPopGenTous[Opérateur].png représente tous les émetteurs de chaque support de chaque support de l'opérateur [Opérateur].

Il créée ensuite dans le dossier /statistiques/emetteur_population_support/StatParOperateurMemeOrdonne 10 graphiques.
Les 8 graphiques qui sont nommé comme ceux créée précedemment utilisient les mêmes données mais la hauteur de l'ordonnée est la même pour tous.
La hauteur est celle de l'opérateur qui possède le plus d'émetteurs toute génération confondus.
Ces graphiques permettent de comparer les graphiques côte à côte.
De plus, il créée encore deux graphiques.
Le premier, statPopgenOperateurGenMax.png représente tous les opérateurs et la générations maximale de leur émetteurs par support sur un même graph.
Le second, statPopgenOperateurTous.png représente tous les opérateurs et tous leur émetteurs par support sur un même graph.
Ces deux dernier graph sont peu lisible c'est pourquoi les 8 premiers sont créée.
